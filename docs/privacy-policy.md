# Privacy Policy

Palabra does not collect, store, or transmit any personal data.

The app works entirely offline. No internet connection is used or required,
and no network permission is declared in the manifest.

Favourites are stored locally in a Room database on the device. They never
leave the device unless the user explicitly exports them using the built-in
export feature, which writes a JSON file to a folder chosen by the user.

No analytics, no crash reporting, no third-party SDKs.
